# TLTO-Net
TLTO-Net: A Trust and Energy-Efficient Routing Framework for Wireless Sensor Networks using Liquid Neural Networks and Termite Life Cycle Optimization
TLTO-Net is a trust-aware routing framework for Wireless Sensor Networks (WSNs) that combines Liquid Neural Networks (LNNs) and Termite Life Cycle Optimization (TLCO) to improve routing efficiency, energy conservation, network security, and communication reliability. It identifies trustworthy nodes, detects abnormal behavior, and selects optimal routes for data transmission.
The project addresses major challenges in WSNs such as high energy consumption, unreliable routing, malicious node behavior, and reduced network lifetime. TLTO-Net improves packet delivery ratio, throughput, trust management, and energy efficiency, making it suitable for applications such as smart cities, healthcare monitoring, environmental monitoring, and Industrial IoT
Users can clone the repository, install the required dependencies (MATLAB/Python), download the WSN-DS dataset, configure simulation parameters, and execute the main simulation file. The framework then performs trust evaluation, route optimization, and network performance analysis automatically
